¿De qué trata?

WisepiloBlogs es una aplicación web para la gestión de blogs personales. Permite a los usuarios registrarse, iniciar sesión, crear, editar y eliminar publicaciones, además de visualizar blogs públicos o personales con un editor de texto enriquecido. El proyecto está desarrollado con Node.js y Express, y sigue una arquitectura modular basada en controladores, modelos y rutas para facilitar el mantenimiento y escalabilidad.

Objetivo

Crear una aplicacion web sencilla pero potente para la escritura y gestión de contenido, para aplicar las funcionalidades de una web centrada en el backend, arquitectura de diseño y distribución de código.

Funcionalidades implementadas

  • Autenticación de usuarios: registro, login y manejo de sesiones con JWT.
  • Gestión de publicaciones: creación, edición y eliminación de entradas de blog.
  • Editor enriquecido: integrado con Quill.js para permitir formato de texto, imágenes y enlaces.
  • Subida de imágenes: manejo de archivos con Multer.
  • Seguridad: sanitización de HTML con sanitize-html para evitar XSS.
  • Contraseñas seguras: encriptadas con Bcryptjs.
  • Gestión de cookies y tokens: mediante cookie-parser y utilidades personalizadas.
  • Validación de datos: mediante esquemas definidos.
