WisepiloGames: Una Colección de Juegos Web

Este proyecto es una colección de tres juegos web interactivos desarrollados desde cero para demostrar proficiencia en JavaScript nativo, diseño modular y patrones de desarrollo. La plataforma fue diseñada con una arquitectura escalable para asegurar componentes reutilizables y un código limpio.

Habilidades

  • Arquitectura Modular (SPA-Like): Implementación de un sistema de carga dinámica (fetch en JS) para reutilizar Header y Footer (foot-head/), simulando un comportamiento de componente moderno y demostrando manejo de la estructura de la aplicación.
  • Lógica de Negocio Compleja (JS Nativo): Desarrollo de la lógica de juego con clases (POO en JS) para separar responsabilidades (Lógica, UI, Control), como se ve en Geo-Grid y Pasapalabra.
  • Manipulación de Datos y APIs: Gestión de archivos JSON como fuente de datos para los juegos y aplicación de lógica de filtrado por dificultad (Geo-Grid). Consumo de APIs externas para la carga dinámica de recursos (banderas).
  • Normalización de Datos y UX: Implementación de una función de normalización de texto para respuestas de usuario (eliminar acentos, espacios, mayúsculas) en Pasapalabra y Geo-Grid, asegurando una mejor experiencia del usuario.
  • Algoritmos y Lógica de IA: Desarrollo de una IA en Tic-Tac-Toe con estrategia de Win-Block-Center que prioriza la victoria y la defensa, mostrando habilidad en la creación de algoritmos de juego.
