Pasapalabra

¿De qué trata?

Inspirado en el clásico juego de televisión Pasapalabra, donde el jugador debe responder correctamente a una serie de preguntas asociadas a cada letra del abecedario, avanzando en la ruleta hasta completar todas las respuestas o agotar el tiempo.

Diseño

Se utilizó un formato basado en clases para manejar las distintas funcionalidades. A diferencia de otros proyectos, aquí no hay una clase Data dedicada a la gestión de datos ni una clase que unifique todas las demás; en su lugar, esa lógica se resolvió mediante funciones. Fue uno de mis primeros proyectos tratando de separar responsabilidades en la web usando clases, por lo tanto no es de los mas claros/entendibles en código.

Algunas implementaciones

Muestra que sabes cargar y transformar datos dinámicamente, en vez de tenerlos “hardcodeados”.


async function loadData(category = 'mix') {
  const res = await fetch('./dataJson.json')
  const data = await res.json()
  return {
    data: extractPromptsFromData(data[category]),
    categories: Object.keys(data)
  }
}

function extractPromptsFromData(dataList) {
  return dataList.map(item => {
    return {
      prompt: item[0],
      answer: item[1]
    }
  })
}

Cómo se validan las respuestas y se avanza en la ruleta.


checkAnswer(answer) {
  answer = this.textNormalize(answer)
  const correctAnswer = this.getCurrentAnswer()
  if (answer === correctAnswer) {
    this.changeStateAnswer(this.answerCorrect)
  } else {
    this.changeStateAnswer(this.answerWrong)
  }
  return {
    index: this.currentPosition,
    state: this.getCurrentState(),
    nextIndex: this.moveToNextUnansweredLetter()
  }
}
